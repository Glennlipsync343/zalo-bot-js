# zalo-bot-js

`zalo-bot-js` là SDK TypeScript cho Zalo Bot API, được thiết kế để dễ dùng với Node.js và phù hợp với người dùng Việt Nam.

## Bạn sẽ tìm thấy gì ở đây

- Cách cài đặt nhanh và tạo `.env`
- Cách kiểm tra token bot
- Cách chạy bot polling để test `/start` và `hello`
- Kiến trúc SDK hiện tại và cách mở rộng sau này
- Ví dụ webhook, examples và test scripts

## Trạng thái hiện tại

SDK hiện đã có:

- `Bot`, `Application`, `ApplicationBuilder`
- `CommandHandler`, `MessageHandler`, `filters`
- `getMe`, `getUpdate`, `sendMessage`, `sendPhoto`, `sendSticker`, `sendChatAction`
- webhook helper cơ bản
- script test thực tế bằng `.env`

Bắt đầu với [Bắt đầu nhanh](./getting-started.md).
