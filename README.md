## 🖥️ What this is

zalo-bot-js is a TypeScript SDK for the Zalo Bot API. It helps you build a bot that can read messages, send replies, and handle events with less work.

This project includes:

- polling support
- webhook helpers
- message handlers
- event handling tools
- TypeScript support

It is made for users who want a clean setup for Zalo bot work on Windows.

## 🧪 Run the project

Use the project in the way that fits your setup.

### Option 1: Polling

Polling checks for new events on a set timer. This works well for simple local testing.

Typical flow:

1. Start the app
2. Connect to the API
3. Read new bot events
4. Send replies based on the handler rules

### Option 2: Webhook

Webhooks send events to your app when something happens. This works well for a server setup.

Typical flow:

1. Set a public endpoint
2. Connect the endpoint to the bot service
3. Listen for incoming events
4. Process each event with your handler
